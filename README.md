這遊戲原本是以前在用VB製作專題時寫的遊戲，現在用網頁復刻，也改良了一些寫法和bug，並支援電腦和部分手機版。(建議用Chrome開啟)

這個劇情全是自己想的，所以可能有點奇怪。而素材大部分是從mnecraft材質包和The Orphanage地圖拿來用的，追逐的音樂是青鬼，部分角色為原創(就是看起來像用小畫家畫的那些)，BGM忘記叫什麼了，以前找的。
